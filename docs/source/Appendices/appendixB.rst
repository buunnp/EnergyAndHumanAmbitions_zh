附录 B 化学基础/复习
========================

本书并不十分依赖以往的化学知识，但了解一些在化石燃料、生物能源和气候变化中发挥作用的基本元素还是很有帮助的。这部分内容可以作为复习，也可以作为对基础知识的初次接触。
